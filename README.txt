# googleAPI_spreadSheet_To_JSON

구글 스프레드시트의 데이터를 JSON파일로 만들어준다.
시작전에 구글콘솔에서 credential파일을 생성후 로컬에서 요청을한다.
인증토큰을 최초생성시 브라우저에 해당 URL을 입력후 Auth인증을 진행하고 해당내용을 token.json 파일로 작성한다.
( 참고 ) https://support.google.com/a/answer/7378726?hl=ko

----

실무에서 마케팅데이터를 정리하는 방식이 엑셀형태였음.
해당데이터를 온라인상으로 이동시켜 고객사에 리포트를 만들어 제공하기 위한 로컬실행파일이다.
리팩토링 및 고도화 필요한데 언제할지 모르겠다.
